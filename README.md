PROCESSO DO TRABALHO LUZ RGB
primeiro foi criado 1 variavel para cada um dos 3 potenciometros
em seguida cada variavel foi direcionada a um pino PWM (de ~11 a ~9) especifico e mapeado para uma faixa de 0 a 255 para que cada potenciometro controle
um aspecto da luz RGB
